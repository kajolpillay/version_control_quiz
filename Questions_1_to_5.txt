Q 1. What are Pull Requests used for?
Pull requests are used for making revisions that then the original developer will decided to accept it into the official project.

Q 3. What is a repo and explain the difference between a local and remote one? 
A repo is a location where all the files for a particular project are store. A local repo is the local git repo stored anywhere on your machine. A remote one are versions of your project that is hosted online and other's can collobrate on it.

Q 4. Where would you find a local repo?
Inside your working directory of your local machine
